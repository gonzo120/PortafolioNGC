Problema

En el problema propuesto por Ringa Tech, se plantea el uso de una red neuronal convolucional pre-entrenada proveniente de TensorFlow Hub. El objetivo inicial es emplear esta red para desarrollar un clasificador capaz de distinguir entre tenedores, cucharas y cuchillos utilizando Python y TensorFlow. Aunque el problema original parecía relativamente sencillo, decidí ir más allá y proponer mi propio desafío. Mi enfoque consistió en crear un clasificador de imágenes que pudiera identificar equipos de fútbol colombianos. Para ello, adapté la red neuronal entrenada, reconfigurando las dimensiones de entrada y salida para trabajar con 21 equipos en lugar de las 3 categorías iniciales. Esta ampliación del problema implicó un mayor nivel de complejidad y requirió modificaciones en el código original proporcionado por Ringa Tech. A lo largo de nuestra interacción, exploramos diversas temáticas relacionadas con la solución propuesta. Abordamos la carga de modelos pre-entrenados desde TensorFlow Hub, la importancia de establecer las dimensiones de entrada adecuadas, así como la aplicación del aumento de datos mediante la función ImageDataGenerator. También destacamos la relevancia de verificar la compatibilidad de versiones y asegurarnos de tener instaladas las dependencias necesarias
